# Hotel PMS AWS Lambda

A serverless MCP server for hotel property management system operations, built
with AWS Lambda and designed to work with Amazon Bedrock AgentCore Gateway for
AI agent integration.

## Features

- **MCP Server**: Model Context Protocol server for AI agent integration
- **Amazon Bedrock AgentCore Gateway**: Native integration with Amazon Bedrock AgentCore
- **Room Management**: Availability checking, pricing, and inventory management
- **Reservation System**: Full reservation lifecycle management
- **Guest Services**: Checkout processing and housekeeping requests
- **Knowledge Base**: Hotel-specific information retrieval via Amazon Bedrock
- **Database Integration**: Aurora Serverless v2 PostgreSQL with automated setup
- **Observability**: Comprehensive logging, tracing, and metrics via AWS Lambda
  Powertools
