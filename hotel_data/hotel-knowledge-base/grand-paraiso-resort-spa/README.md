# Grand Paraíso Resort & Spa - Base de Conocimientos

Este directorio contiene la base de conocimientos para Grand Paraíso Resort &
Spa, organizada como archivos FAQ en markdown que cubren todos los aspectos de
las operaciones del hotel y servicios para huéspedes.

## Estructura

- `informacion-general.md` - Información general e identificación
- `ubicacion-contacto.md` - Ubicación e información de contacto
- `habitaciones-suites.md` - Detalles de habitaciones y suites
- `gastronomia.md` - Información de restaurantes y bares
- `instalaciones.md` - Instalaciones y amenidades
- `politicas-servicios.md` - Políticas del hotel y servicios
- `operaciones-rendimiento.md` - Datos operacionales y métricas de rendimiento
